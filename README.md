# Cafe-Maker
This is a game made with pygame.

-All images are made by me.
-Using Jupyter Notebook.

이 프로젝트는 `pygame`을 사용하여 커피 및 음료 제조 시뮬레이션 게임을 구현한 것입니다. 주요 내용은 다음과 같습니다:

1. **게임 초기화 및 설정**:
    - `pygame`을 초기화하고 게임 화면 크기 및 기타 기본 설정을 합니다.
    - 다양한 이미지(버튼, 음료 아이콘, 카테고리 이미지 등)을 로드하여 게임에 사용할 준비를 합니다.
2. **버튼 클래스** (`Button`):
    - 버튼을 클릭할 수 있도록 관리하는 클래스로, 버튼의 이미지와 위치, 클릭 시 호출할 함수를 정의합니다. 또한 버튼 위에 텍스트도 추가할 수 있습니다.
3. **재료 클래스** (`Ingredient`):
    - 음료에 필요한 재료(물, 우유, 커피 등)를 추적하는 클래스입니다. 각 재료는 `Add` 메서드를 통해 수량을 추가할 수 있습니다.
4. **화면 전환 함수**:
    - 다양한 카테고리 화면(과일, 우유, 커피, 시럽 등)을 보여주는 함수들이 있으며, 사용자가 버튼을 클릭하여 해당 카테고리에서 재료를 선택할 수 있습니다.
    - `FruitCategoryScreen`, `MilkCategoryScreen`, `CoffeeCategoryScreen`, `PowderCategoryScreen`, `SyrupCategoryScreen` 등의 함수가 재료를 선택하도록 유도합니다.
5. **게임 화면 및 인터페이스**:
    - `playScreen` 함수는 실제 게임 화면을 구성합니다. 여기에서 사용자는 음료 제조를 위해 재료를 선택하고, 제조가 완료되면 점수를 얻습니다.
    - 사용자가 선택한 재료 목록을 보여주는 기능도 포함되어 있으며, 점수와 주문된 음료 정보를 실시간으로 업데이트합니다.
6. **게임 로직**:
    - 게임이 진행되는 동안 사용자는 재료를 추가하여 특정 음료를 만들도록 해야 합니다. 예를 들어, '아메리카노'는 '물'과 '커피'로 만들어지며, 재료가 정확히 맞으면 음료가 완성되어 점수가 증가합니다.
    - `Complete` 함수에서 음료가 완성되었는지 체크하고, 완료된 음료에 맞는 재료가 추가되었는지 검증합니다.
7. **게임 종료 및 초기 화면**:
    - 게임 종료 시 `quitgame()` 함수를 통해 게임을 종료할 수 있습니다.
    - `TitleScreen` 함수는 게임 시작 화면을 나타내며, "시작" 버튼을 눌러 게임을 시작할 수 있습니다.

이 게임은 음료 제조 시뮬레이션을 통해 사용자가 음료를 만들고, 그 과정을 게임처럼 즐길 수 있도록 만든 프로젝트입니다. 각 재료의 수량과 조합에 따라 음료를 성공적으로 만들면 점수를 얻고, 여러 음료를 만드는 과정을 반복하며 진행됩니다.
